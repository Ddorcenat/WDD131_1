Overview

Purpose

This website serves as a digital cookbook that allows users to browse a curated collection of delicious recipes. It aims to solve the problem of organizing scattered recipes by providing an interactive, searchable, and responsive platform where users can filter meals by category and adjust serving sizes dynamically.

Audience

The target audience includes home cooks of all skill levels, from busy students looking for quick meals to food enthusiasts seeking new dinner ideas. They are likely accessing the site from mobile devices while in the kitchen or desktops while meal planning, so they value clear readability and easy navigation.

Dynamic elements

I will use JavaScript to make the site interactive. The recipe data will be stored in an Array of Objects within an ES Module. I will use the .map() method to dynamically render the recipe cards to the DOM. I will use event listeners on filter buttons to trigger a .filter() method (conditional branching) to show specific categories. I will also listen for changes on an input field to dynamically calculate ingredient amounts based on servings.
